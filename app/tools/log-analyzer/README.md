# Log Analyzer

Parse mixed log text and produce operational summaries.

## Features

- Access log, JSON log, and freeform parsing
- Level and HTTP status class aggregation
- Top paths and recent errors

## Parameters

- Log text input
- Top rows limit

## Usage

- Paste logs from files or terminal output
- Run analysis to inspect error and traffic hotspots

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- Log input changes create debounced entries
- Display params update latest history params
