# HTML Encoder/Decoder

Encode plain text into HTML entities or decode HTML entities back to readable text.

## Features
- Two-way editable panes for plain text and HTML entities
- URL-synced input side and history restoration
- Oversized input warnings when URL sync exceeds 2 KB

## Parameters
- Input side (plain text or HTML)
- Plain text input
- HTML input

## URL State
- Input side and its value are synced to the URL
- Output-only value is excluded from URL sync

## History
- Input changes create valued entries
- Active input side is saved with each entry
