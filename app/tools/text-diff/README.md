# Text Diff Viewer

Compare two text inputs with a unified diff view.

## Features
- Unified diff output with highlights
- File upload support
- Fullscreen mode

## Parameters
- View options (if available)

## Usage
- Paste or upload text on both sides to compare

## URL State
- Inputs and parameters sync to the URL query

## History
- Input changes create history entries
- Parameter changes update the latest entry
