# JSON Diff Viewer

Compare two JSON inputs and view differences in table or text mode.

## Features
- Table view and unified text view
- File upload support
- Error highlighting

## Parameters
- View mode (table/text)

## Usage
- Paste or upload JSON in both panes
- Switch views to inspect differences

## URL State
- Inputs and parameters sync to the URL query

## History
- Input changes create history entries
- Parameter changes update the latest entry
