# SVG to PNG Converter

Convert SVG files to PNG format with customizable dimensions and background colors.

## Features

- **File Upload**: Upload SVG files directly from your device
- **Code Editor**: Paste or edit SVG code with syntax highlighting
- **Real-time Preview**: See your SVG rendered instantly as you edit
- **Custom Dimensions**: Set exact width and height for PNG output (1-8192px)
- **Aspect Ratio Lock**: Maintain proportions when resizing
- **Background Control**: Set transparent or custom color backgrounds
- **Quick Size Presets**: Common sizes (16, 32, 64, 128, 256, 512, 1024px)
- **PNG Download**: Export your SVG as a high-quality PNG file

## How It Works

1. Upload an SVG file or paste SVG code directly
2. Edit the SVG code if needed - preview updates in real-time
3. Set desired output dimensions (width/height)
4. Choose background color (transparent or custom)
5. Click "Download PNG" to save the converted image

## Technical Details

- Uses HTML5 Canvas API for conversion
- Supports all standard SVG features
- Preserves transparency when background is set to transparent
- Maximum output size: 8192 x 8192 pixels
- Output format: PNG (lossless)

## Use Cases

- Converting icons for different platforms
- Creating social media images from vector graphics
- Generating favicons and app icons
- Batch processing SVG assets
- Quick SVG debugging and visualization
