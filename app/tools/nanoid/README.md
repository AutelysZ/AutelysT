# NanoID

Generate NanoIDs with custom alphabets, lengths, and batch sizes.

## Features

- Generate multiple NanoIDs at once
- Preset alphabets plus custom alphabet input
- Copy and download generated IDs
- Alphabet and entropy stats

## Parameters

- Length
- Count
- Alphabet preset
- Alphabet characters

## Usage

- Adjust length, count, and alphabet settings, then click Generate

## URL State

- Inputs and parameters are synced to the URL hash for sharing
- Output-only fields are excluded from URL sync

## History

- Generated ID lists are stored as history entries
- Parameter changes update the latest entry
