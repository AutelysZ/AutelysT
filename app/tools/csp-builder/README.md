# CSP Builder

Build and edit Content-Security-Policy headers with directive-aware editing.

## Features
- Parse CSP policies into directives and values
- Edit directives and regenerate a normalized policy output
- URL-synced policy input with history restore
- Copy-ready policy value and full header line

## Parameters
- Policy text

## URL State
- Policy text is synced to the URL
- Output-only header text is excluded from URL sync

## History
- Policy edits create history entries
