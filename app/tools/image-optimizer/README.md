# Image Optimizer

Resize, compress, and convert images in-browser.

## Features

- JPEG/PNG/WebP/AVIF output formats
- Quality and max dimension controls
- Before/after preview with size comparison

## Parameters

- Output format
- Quality
- Max width and max height

## Usage

- Upload an image
- Configure optimization settings
- Optimize and download output

## URL State

- Parameters sync with URL state (hash-based sharing)
- Binary image content is not stored in URL state

## History

- Uploaded filename snapshots create history entries
- Parameter changes update latest history params
