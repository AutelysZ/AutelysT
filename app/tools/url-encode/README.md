# URL Encoder/Decoder

Encode and decode URLs with detailed parsing.

## Features
- Responsive layout on mobile
- Dual-pane encode/decode
- URL component breakdown
- Search and hash param parsing

## Parameters
- None

## Usage
- Type in either pane to encode/decode

## URL State
- Inputs sync to the URL query

## History
- Input changes create history entries
- Parameter changes update the latest entry
