# XPath

Evaluate XPath expressions against XML/HTML content.

## Features

- Node set, string, number, and boolean result modes
- XML input with XPath editor
- Formatted output rendering

## Parameters

- XML/HTML input
- XPath expression
- Return type

## Usage

- Paste XML/HTML
- Enter XPath expression
- Select return type and evaluate

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- XML/expression changes create debounced entries
- Return type updates latest history params
