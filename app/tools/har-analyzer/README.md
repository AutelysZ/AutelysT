# HAR Analyzer

Analyze HAR files for performance and traffic insights.

## Features

- Total request/time/transfer summary
- Method and status bucket breakdown
- Slowest request list

## Parameters

- HAR JSON input
- Top rows limit

## Usage

- Paste HAR JSON
- Run analysis and inspect summary panels

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- HAR input changes create debounced entries
- Display params update latest history params
