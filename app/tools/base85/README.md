# Base85 / Base91

Encode and decode Base85 (Ascii85/Z85) and Base91 strings.

## Features

- Dual-pane encode/decode workflow
- Ascii85 (framed/raw), Z85, and Base91 variants
- Text encoding selection
- File upload and download

## Parameters

- Text encoding
- Encoding variant (Ascii85, Z85, Base91)

## Usage

- Type in either pane to convert between plain text and the selected encoding

## URL State

- Inputs and parameters are synced to the URL hash for sharing
- Output-only fields are excluded from URL sync

## History

- Input edits create history entries
- Parameter changes update the latest entry
