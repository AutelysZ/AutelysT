# License Generator

Generate software licenses with a guided questionnaire or manual selection, then export ready-to-use license text.

## Features

- Guided questionnaire to recommend a license
- Manual selection from popular SPDX licenses
- Auto-fill placeholders (year, holder, project, email, website)
- Copy or download the generated license text

## Parameters

- Step index
- Selection mode (guided/manual)
- Selected license ID
- Questionnaire answers
- License details (year, holder, project, email, website)

## URL State

- All inputs and selections are synced to the URL
- Output text is not synced

## History

- Input changes create history entries
- Parameter changes update history params without creating new entries
