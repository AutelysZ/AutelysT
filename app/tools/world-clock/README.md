# World Clock

Track time across multiple time zones using a live clock or a custom reference time.

## Features

- Live clock updates every second
- Custom reference time with date/time picker
- Add/remove multiple time zones

## Parameters

- Mode (live or custom)
- Base time (custom reference)
- Time zone list

## URL State

- Mode, base time, and time zones sync to the URL

## History

- Custom time edits create valued entries
- Mode and time zone changes update the latest entry
