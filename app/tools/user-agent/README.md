# User Agent Parser & Builder

Parse and build User-Agent strings with a dual-panel editor, plus JSON and table views for structured editing.

## Features

- Parse User-Agent strings into structured JSON
- Build User-Agent strings from editable fields
- Toggle between JSON and table views
- URL-synced inputs with history support

## Parameters

- Left panel: User-Agent string input
- Right panel: JSON or table view
- Active side determines parse/build direction

## URL State

- Both panel inputs and view selection sync to URL

## History

- Active input creates history entries
- Parsed state is restored when loading history
