# CUID2

Generate and validate CUID2 identifiers.

## Features

- Batch CUID2 generation
- Configurable output length and fingerprint
- Multi-line validation summary

## Parameters

- Count
- Length
- Fingerprint
- Validation input text

## Usage

- Configure generator settings and generate IDs
- Paste IDs into validator textarea

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- Validator input changes create debounced entries
- Generator params update latest history params
