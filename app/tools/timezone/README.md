# Time Zone Converter

Convert times between time zones, including Unix epoch formats.

## Features
- Dual-pane conversion
- Date/time picker
- IANA timezone support

## Parameters
- Left timezone
- Right timezone

## Usage
- Enter a time on either side to convert

## URL State
- Inputs and parameters sync to the URL query

## History
- Input changes create history entries
- Parameter changes update the latest entry
