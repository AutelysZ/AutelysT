# MIME Lookup

Look up MIME types by filename and list known extensions for MIME types.

## Features

- Detect MIME types from filenames or paths
- List known extensions for detected MIME types
- Look up extensions from a MIME type string with autocomplete
- Upload or drag-and-drop a file to detect MIME from filename extensions
- URL-synced inputs with history restore

## Parameters

- Filename
- MIME type

## URL State

- Filename and MIME type are synced to the URL

## History

- Changing inputs creates history entries
- Selecting a history entry restores inputs and updates the URL
