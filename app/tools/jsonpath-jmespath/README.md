# JSONPath / JMESPath

Evaluate JSON queries with either JSONPath or JMESPath.

## Features

- Switchable JSONPath and JMESPath engines
- JSON + query editor
- Formatted output rendering

## Parameters

- Query engine
- JSON input
- Query expression

## Usage

- Paste JSON and a query
- Choose engine and evaluate

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- JSON/query changes create debounced entries
- Engine changes update latest history params
