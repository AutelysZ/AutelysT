# SemVer

Semantic versioning helper for compare, range evaluation, incrementing, and sorting.

## Features

- Compare two semantic versions
- Check if a version satisfies a range
- Increment versions by release type
- Sort multiline version lists and flag invalid entries

## Parameters

- Compare inputs, range inputs, increment inputs
- Release type
- Sort direction

## Usage

- Fill any section independently
- Results update as inputs change

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- Text input changes create debounced entries
- Parameter changes update latest history params
