# Mermaid Renderer

Render Mermaid diagram syntax to SVG and PNG.

## Features

- Live Mermaid rendering
- Theme selection
- SVG and PNG download

## Parameters

- Mermaid source text
- Theme
- PNG scale

## Usage

- Paste Mermaid syntax
- Render SVG preview
- Generate and download PNG if needed

## URL State

- Inputs/params sync with URL state (hash-based sharing)

## History

- Source changes create debounced entries
- Theme/scale changes update latest history params
